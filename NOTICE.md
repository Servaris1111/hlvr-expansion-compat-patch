# Notice

This repository contains only compatibility patch code, installer scripts, documentation, and small proxy DLL binaries built from this project's source.

It does not contain Half-Life, Opposing Force, Blue Shift, Half-Life VR Mod, Steam, Valve, or Gearbox game assets.

Users must own/install the relevant Steam games and mods. The installer copies expansion content from the user's local Steam installation.

HLFixes is downloaded from the official IntriguingTiles/HLFixes GitHub release page at install time.
